**1. 버그 설명**

버그가 무엇인지에 대해 명확하고 간결하게 설명합니다.

**2. 버그 재현 설명**

동작을 재현하기 위해선 아래와 같이 합니다.:
1. '...'로 이동
2. '....'를 클릭
3. '....'까지 스크롤
4. 오류 보기

**3. 예상되는 동작**

예상되는 일에 대한 대해 명확하고 간결하게 설명합니다.

**4. 스크린샷**

해당되는 경우, 문제를 설명하는데 도움이 되는 스크린샷을 추가해주세요.

**PC (다음 정보를 작성해 주세요):**

 - OS: [e.g. macOS]
 - Version [e.g. Ventura]
 - Browser [e.g. Chrome]

**스마트폰 (다음 정보를 작성해 주세요):**

 - Device: [e.g. iPhone14]
 - OS: [e.g. iOS16.4]
 - Browser [e.g. Safari]

**5. 추가 의견**

문제에 대한 의견 또는 추가 의견을 설명합니다.
